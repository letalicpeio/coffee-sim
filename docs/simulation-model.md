# Simulation Model — Coffee-Sim

Coffee-Sim estimates flavour characteristics based on brewing parameters.

The goal is educational exploration of coffee extraction.

The model is heuristic and simplified.

It is not intended to represent exact scientific measurements.

---

# Output Attributes

The simulator estimates attributes such as:

acidity  
sweetness  
bitterness  
body  
balance

These values are visualised in the radar chart.

---
